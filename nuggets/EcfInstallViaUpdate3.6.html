ECF 3.6.1 Install via p2 repository

For a detailed step-by-step description of how to install OSGi Remote Services into Eclipse see Installing Into Eclipse. For a step-by-step description of how to add OSGi Remote Services into your Target Platform see Add to Target Platform.

  • In Eclipse, choose Help -> Install New Software...
  • Click on the Add button on the right.
  • Type in a Name in the textbox. For example, type: ECF 3.6.0.
  • Paste the following URL into the Location field.

    http://download.eclipse.org/rt/ecf/3.6.1/site.p2
  • Click OK.
  • Select the Eclipse Communication Framework (ECF) check box only. NOTE: This check box will result in including 2 sub-items:
    • Eclipse Communication Framework (ECF)
      1. ECF Target Components for Eclipse
      2. Source for ECF Target Components for Eclipse
  • NOTE: Only this first feature and it's two sub-items should be selected for install into Eclipse. Explanation: The ECF Repo has three top-level features:
    1. Eclipse Communication Framework (ECF)
    2. Eclipse Communication Framework File Transfer Features (ECF)
    3. Eclipse Communication Framework Remote Services (ECF)
    The Eclipse Communication Framework File Transfer Features (ECF) and Eclipse Communication Framework Remote Services Features (ECF) are already included in the ECF Target Components for Eclipse and so should not be checked separately. These two features are intended for use in non-Eclipse OSGi environments (e.g. servers) and will conflict with install into Eclipse.
  • Click on Next.
  • Follow through the remaining dialogs to complete the installation.

To uninstall ECF SDK...

  • Choose Help -> Install New Software...
  • Click on already installed (lower part of install manager dialog)
  • Click on Installed Software tab
  • Click on ECF Target Components
  • Click on Uninstall ...
