Anonymous CVS Access to ECF Source Code

To load ECF source code into your workspace

  1. Get an Eclipse version 3.3 or 3.4 build (http://download.eclipse.org/eclipse/downloads/)
  2. Create a CVS repository location for

    :pserver:anonymous@dev.eclipse.org:/cvsroot/technology/ecf

    Hint: Select the bold text, open the CVS Repositories view, and select Paste Connection or hit Ctrl+V to add the connection
  3. Expand HEAD > org.eclipse.ecf > builds > plugins
  4. Checkout project: org.eclipse.ecf.releng
  5. Import the projectSet-anonymous.psf project set using File > Import > Team Project Set in the releng project you just checked out. If you are an ECF committer, and able to make an extssh connection, you can use projectSet-extssh.psf.

Anonymous CVS server info

  • Host: dev.eclipse.org
  • Repository Path: /cvsroot/technology
  • User: anonymous
  • Password: <empty>
  • Connection method: pserver
  • Module (within org.eclipse.ecf): plugins, providers, incubator/plugins, tests

    Instructions for Accessing via Anonymous CVS

  1. Within Eclipse, chooose File->Import...->CVS->Projects from CVS (or open CVS Repositories view)
  2. Select repository entered above and navigate HEAD > org.eclipse.ecf > plugins > providers > incubator > plugins, and > tests modules
  3. Select desired plugins, choose Next, and choose Checkout into Workspace As Projects and Finish
