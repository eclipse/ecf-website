Anonymous Access to ECF Source Code

To load ECF source code into your workspace

  1. Get a recent Eclipse version (http://download.eclipse.org/eclipse/downloads/)
    1. If not present, install egit via p2 (http://www.eclipse.org/egit/)
    2. Clone the git repository from the view menu "Clone a git repository and add the clone to this view"

      git://git.eclipse.org/gitroot/ecf/org.eclipse.ecf.git

    3. Import all projects from the context menu of the newly created git repo, using Import Project....

      If you are an ECF committer, and able to use a ssh connection instead of git.

    Anonymous git server info

    • Host: git.eclipse.org
    • Repository Path: /gitroot/ecf/org.eclipse.ecf.git
    • Connection method/protocol: git (or ssh for committers)
