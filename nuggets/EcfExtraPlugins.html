ECF Github Repositories

ECF has a Github organization with many repositories, including those for OSGi Remote Management, the following distribution providers:

  • TCP Socket Distribution Provider
  • JMS/ActiveMQ
  • Bndtools 4 workspace for Remote Services development
  • Jax-RS Providers (Jersey and/or Apache CXF)
  • An MQTT Provider (based upon Eclipse Paho MQTT)
  • Google RPC provider
  • XML-RPC provider
  • A provider based upon JavaGroups
  • Etcd-based Discovery provider
  • Hazelcast Provider
along with an extension to the r-OSGi provider to use Websockets, a Java-only (no Bundle layer) Service Registry...optionally with ECF Remote Services/RSA, some example Raspberry Pi projects, and an example client OSGi Remote Service built on the public Geonames REST/JSON API at geonames.org.
