ECF 3.3/Helios Install -- for use with Eclipse Helios (3.6) or Galileo (3.5.X)

Note: ECF is part of the Helios Release Train, and as such is available via Helios repository, which is built into Eclipse. You can access this repository by going to Help -> Install New Software...->Work With (select Helios from combo box). Then under Eclipse Runtime Target Platform Components find Eclipse Communication Framework Target Components to install.

Below is the procedure to install ECF via the ECF Repository directly (rather than the Helios repository as described above).

  • In Eclipse, choose Help -> Install New Software...
  • Click on the Add button on the right.
  • Type in a Name in the textbox. For example, type: ECF 3.3/Helios.
  • Paste the following URL into the Location field.
    http://download.eclipse.org/rt/ecf/3.3/helios/site.p2
  • Click OK.
  • Select the Eclipse Communication Framework (ECF) check box. NOTE: This check box will result in including 4 sub-items
    1. ECF 3.3 Patch for Eclipse 3.5.X
    2. Eclipse Communication Framework Target Components
    3. Source for ECF 3.3 Patch for Eclipse 3.5.X
    4. Source for Eclipse Communication Framework Target Components
    Under normal conditions, all four of these sub-items should be included/selected for install.. The ECF 3.3 Patch for Eclipse 3.5.X is necessary because part of ECF is included with the Eclipse platform itself, and this patch will update these relatively old bundles that were installed with Eclipse 3.5.X. The source code for the ECF can be removed, but *both* the ECF 3.3 Patch for Eclipse 3.5.X and the Eclipse Communication Framework Target Components must be installed to properly use ECF 3.3
  • Click on Next.
  • Follow through the remaining dialogs to complete the installation.

To uninstall ECF ...

  • Choose Help -> Install New Software...
  • Click on already installed
  • Click on Installed Software
  • Click on Eclipse Communication Framework Target Components
  • Click on Uninstall ...
