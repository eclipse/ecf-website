ECF 3.4.0 Install via p2 repository/Eclipse update site

  • In Eclipse, choose Help -> Install New Software...
  • Click on the Add button on the right.
  • Type in a Name in the textbox. For example, type: ECF 3.4.
  • Paste the following URL into the Location field.
    http://download.eclipse.org/rt/ecf/3.4/site.p2
  • Click OK.
  • Select the Eclipse Communication Framework (ECF) check box. NOTE: This check box will result in including 4 sub-items
    1. ECF 3.4 Patch for Eclipse 3.5-3.6
    2. Eclipse Communication Framework Target Components
    3. Source for ECF 3.4 Patch for Eclipse 3.5-3.6
    4. Source for Eclipse Communication Framework Target Components
    If you are using Helios (3.6.X version of Eclipse), the ECF 3.4 Patch for Eclipse 3.5-3.6 is not necessary. This patch is only necessary if you are installing ECF into some Galileo version (3.5.X) of Eclipse.
  • Click on Next.
  • Follow through the remaining dialogs to complete the installation.

To uninstall ECF SDK...

  • Choose Help -> Install New Software...
  • Click on already installed
  • Click on Installed Software
  • Click on Eclipse Communication Framework Target Components
  • Click on Uninstall ...
