ECF 3.2 Install -- for use with Eclipse Galileo (3.5.X) or Helios (3.6)

  • In Eclipse, choose Help -> Install New Software...
  • Click on the Add button on the right.
  • Type in a Name in the textbox. For example, type: ECF 3.2 SDK.
  • Type the following URL into the Location field.
    http://download.eclipse.org/rt/ecf/3.2/3.6/site.p2
  • Click OK.
  • Select the Eclipse Communication Framework (ECF) check box. NOTE: This check box will result in including 4 sub-items
    1. ECF 3.2 Patch for Eclipse 3.5.X
    2. Eclipse Communication Framework SDK
    3. Source for ECF 3.2 Patch for Eclipse 3.5.X
    4. Source for Eclipse Communication Framework SDK
    . All four of these sub-items should be included/selected for install.. The ECF 3.2 Patch for Eclipse 3.5.X is necessary because part of ECF is included with the Eclipse platform itself, and this patch will update these relatively old bundles that were installed with Eclipse 3.5.X. The source code for the ECF can be removed, but *both* the ECF 3.2 Patch for Eclipse 3.5.X and the Eclipse Communication Framework SDK must be installed to properly use ECF 3.2
  • Click on Next.
  • Follow through the remaining dialogs to complete the installation.

To uninstall ECF ...

  • Choose Help -> Install New Software...
  • Click on already installed
  • Click on Installed Software
  • Click on Eclipse Communication Framework (SDK)
  • Click on Uninstall ...
