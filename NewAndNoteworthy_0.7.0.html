ECF API Additions and Changes


VOIP Call Setup API (experimental)


A new experimental API for doing VOIP call setup has been introduced in plugin org.eclipse.ecf.call. This API is still under development, and no provider implementations yet exist that implement the ICallContainer entry point adapter interface. The initial API is being released to get community feedback to guide the further development of the API and the selection/ordering of provider implementations.

See the javadocs for the org.eclipse.ecf.call, and the org.eclipse.ecf.call.events

Please provide any thoughts/comments/feedback to either the ecf-dev@eclipse.org mailing list or the eclipse.technology.ecf newsgroup. For access to these forums see here.
