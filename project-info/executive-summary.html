Eclipse Communication Framework (ECF) has completed and deployed v0.9.3.

We are ready for an API freeze on the core APIs, and are now focussing on

  • Creating more/better user interface extension points to support easy creation of clients
  • Building more provider implementations to support other protocols
  • Addressing API deficiencies found through community usage and project team testing
  • Creating more documentation for the project in the form of tutorials, all APIs, and example applications

ECF is participating in the 2007 Europa Simultaneous Release
