 About the ECF Project

ECF is a framework designed to simplify the development of distributed Eclipse-based plugins and applications. The framework can be used to create plugins, tools, or full Eclipse RCP applications that require asynchronous messaging and communications.

ECF also provides a unique open distributed component model based upon the Equinox runtime, along with a growing set of open, extensible communications components. All APIs, components, and protocol implementations may be reused, extended, or replaced by developers.

To test and demonstrate the framework, we have also built some example applications with features such as user presence, instant messaging, multi-user chat, file sharing, co-browsing, shared drawing/whiteboard, and collaborative text editing. See plans and subprojects to see subprojects currently underway.

ECF is currently in incubation. We anticipate having our 1.0 release in early October, 2006.
