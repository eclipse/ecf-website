IRC Provider

IRC Provider Approved for Distribution under EPL


The IRC provider source code now available via the ECF eclipse.org CVS repository. Previously, this code was not available because it the library upon which it depends was not approved by Eclipse Foundation Legal. The library has now been approved (see ECF IP log. See here for instructions on how to get CVS access to the this IRC provider (plugin org.eclipse.ecf.provider.irc). Note that the IRC provider is not yet part of the build, so to get/use the IRC provider without building yourself see the ECF Extras site here
